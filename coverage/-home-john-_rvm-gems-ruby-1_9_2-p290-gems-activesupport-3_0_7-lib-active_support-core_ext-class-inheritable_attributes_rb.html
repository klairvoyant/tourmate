12 # Allows attributes to be shared within an inheritance hierarchy. Each descendant gets a copy of
 
13 # their parents' attributes, instead of just a pointer to the same. This means that the child can add elements
 
14 # to, for example, an array without those additions being shared with either their parent, siblings, or
 
15 # children. This is unlike the regular class-level attributes that are shared across the entire hierarchy.
 
16 #
 
17 # The copies of inheritable parent attributes are added to subclasses when they are created, via the
 
18 # +inherited+ hook.

21 #    class_inheritable_accessor :hair_colors
