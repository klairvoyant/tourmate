61       # A new (class) method will be created with the given name. If a value is

63       # Otherwise, the given block will be used to compute the value of the

350     # Allows access to the object attributes, which are held in the
 
351     # <tt>@attributes</tt> hash, as though they were first-class methods. So a
 
352     # Person class with a name attribute can use Person#name and Person#name=
 
353     # and never directly use the attributes hash -- except for multiple assigns
 
354     # with ActiveRecord#attributes=. A Milestone class can also ask
 
355     # Milestone#completed? to test that the completed attribute is not +nil+
 
356     # or 0.
 
357     #
 
358     # It's also possible to instantiate related objects, so a Client class
 
359     # belonging to the clients table with a +master_id+ foreign key can
 
360     # instantiate master through Client#master.
 
361     def method_missing(method_id, *args, &block)
 
362       method_name = method_id.to_s
 
363       if match = match_attribute_method?(method_name)
 
364         guard_private_attribute_method!(method_name, args)
 
365         return __send__(match.target, match.attr_name, *args, &block)
 
366       end
 
367       super
 
368     end
